  2. Multiple answer:

    In the following Java program, which methods are class methods (choose all that apply).

    public class Foo { 
        public static void main(String[] args) { 
            System.out.println(new Foo()); 
        } 
        public String toString() { 
            return "BAR"; 
        } 
        public static int flub(int x) { 
            return 2*x + 1; 
        } 
        public int embiggen(int y) { 
            return y+1; 
        } 
    }

    1. Foo

    2. main

    3. toString

    4. flub

    5. embiggen
